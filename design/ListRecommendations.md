# List Recommendations

Client will invoke the Kruize listRecommendations API To get the recommendations generated by kruize for a particular experiment or all experiments. Documentation still in progress stay
tuned.

# Attributes


* `experiment_name` \
  A unique string name is specified for identifying individual experiment, if not provided this API returns the recommendations for all experiments

# Example input

```
http://192.168.49.2:31229/listRecommendations?experiment_name="quarkus-resteasy-autotune-min-http-response-time-db"
```
